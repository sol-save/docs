## Intro

You can use gitsol in preinitialised as well as new repos. If the current directory isn't a git repo already, gitsol will go ahead and run `git init` for you. 
To initialize a new repo, run the gitsol init command

```bash
gitsol init
```

or

```bash
npx gitsol init
```

Every time you run this `gitsol init` command, the CLI automatically creates a new repo on the gitsol contract using the gitsol keypair you funded on the previous page.
