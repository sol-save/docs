# Get Started

## Quick Start with Npm

Install it globally using npm

```bash
npm install -g gitsol
```

## Using npx

```bash
npx gitsol
```

## Setup up your account

Run the gitsol command

```bash
gitsol
```

Type in your pass word or leave it unencrypted<br /><br />

<br />
<br />

Next get an api key from [web3.storage](https://web3.storage/) and enter it, it's free.<br /><br />

<br />

`solana airdrop 2 <address>`
